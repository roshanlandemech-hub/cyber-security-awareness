<section id="case-studies">

  <h2>Real World Cybersecurity Case Studies</h2>

 

  <button class="accordion">🎯 WannaCry Ransomware Attack (2017)</button>

  <div class="panel">

    <p>The WannaCry ransomware affected over 200,000 systems globally across 150 countries, encrypting user data and demanding Bitcoin payments for decryption. It targeted computers running Microsoft Windows using an exploit called EternalBlue. The NHS (UK) and major corporations like FedEx were heavily impacted. Lesson: Always update your OS and maintain regular data backups to minimize risks.</p>

  </div>

 

  <button class="accordion">🔓 Facebook Data Breach (2019)</button>

  <div class="panel">

    <p>In 2019, over 540 million Facebook user records were exposed due to poorly configured cloud servers. Information such as account IDs, comments, and reactions were accessible to anyone online. The breach raised serious questions about cloud security and data privacy. Lesson: Always secure cloud databases and monitor data-sharing permissions regularly.</p>

  </div>

 

  <button class="accordion">💳 Equifax Data Breach (2017)</button>

  <div class="panel">

    <p>Equifax, one of the largest credit reporting agencies, suffered a massive data breach that exposed personal information of 147 million people. The attack was caused by an unpatched vulnerability in Apache Struts. Lesson: Keep software up-to-date, use proper encryption, and have a strong incident response plan.</p>

  </div>

 

  <button class="accordion">💻 SolarWinds Supply Chain Attack (2020)</button>

  <div class="panel">

    <p>This attack targeted the SolarWinds Orion software, compromising major U.S. government agencies and Fortune 500 companies. Hackers inserted malicious code into legitimate software updates, giving them backdoor access. Lesson: Regularly audit supply chains, use code integrity checks, and monitor network traffic for anomalies.</p>

  </div>

 

  <button class="accordion">🧾 Colonial Pipeline Ransomware (2021)</button>

  <div class="panel">

    <p>The Colonial Pipeline, a major U.S. fuel supplier, was forced to shut down operations after a ransomware attack by the DarkSide group. The company paid a $4.4 million ransom in Bitcoin to regain control. Lesson: Implement network segmentation, strong password policies, and conduct regular cyber drills.</p>

  </div>

</section>

  const passwordInput = document.getElementById("passwordInput");

  const strengthBar = document.getElementById("strengthBar");

  const strengthText = document.getElementById("strengthText");

  passwordInput.addEventListener("input", () => {

    const val = passwordInput.value;

    let strength = 0;

    if (val.match(/[a-z]+/)) strength++;

    if (val.match(/[A-Z]+/)) strength++;

    if (val.match(/[0-9]+/)) strength++;

    if (val.match(/[$@#&!]+/)) strength++;

    if (val.length > 8) strength++;

 

    switch (strength) {

      case 0:

        strengthBar.style.width = "0%";

        strengthBar.style.background = "transparent";

        strengthText.textContent = "Strength: None";

        break;

      case 1:

        strengthBar.style.width = "20%";

        strengthBar.style.background = "red";

        strengthText.textContent = "Strength: Very Weak";

        break;

      case 2:

        strengthBar.style.width = "40%";

        strengthBar.style.background = "orange";

        strengthText.textContent = "Strength: Weak";

        break;

      case 3:

        strengthBar.style.width = "60%";

        strengthBar.style.background = "yellow";

        strengthText.textContent = "Strength: Moderate";

        break;

      case 4:

        strengthBar.style.width = "80%";

        strengthBar.style.background = "lightgreen";

        strengthText.textContent = "Strength: Strong";

        break;

      case 5:

        strengthBar.style.width = "100%";

        strengthBar.style.background = "#00ff00";

        strengthText.textContent = "Strength: Very Strong";

        break;

    }

  });
